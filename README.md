# 智慧树掌握度 AI 自动答题脚本

用于智慧树掌握度练习的浏览器脚本。脚本会自动进入需要提升的知识点，调用 AI 完成答题，并在解析页沉淀参考答案，让后续重复题目优先使用已验证答案。

## 适用范围

- 智慧树掌握度练习页面
- 单选题、多选题、判断题、填空客观题
- Tampermonkey 或其他兼容用户脚本管理器

## 快速开始

1. 安装 Tampermonkey 浏览器扩展。
2. 新建用户脚本。
3. 将 `main.user.js` 的全部内容复制进去并保存。
4. 打开智慧树课程掌握度页面。
5. 点击页面右侧的 `AI` 悬浮按钮。
6. 选择接口模式后，点击「开始自动答题」。

默认使用免费模型，不需要额外配置。需要使用自己的模型时，切换到「自定义」并填写 `Base URL`、`API Key` 和 `Model`，然后点击「保存接口设置」。

## 运行流程

脚本启动后会按顺序执行：

1. 查找掌握度未满的知识点。
2. 优先进入掌握度最低的知识点。
3. 点击「去提升」进入练习。
4. 识别题目内容并生成答案。
5. 完成后提交练习。
6. 进入解析页采集参考答案。
7. 返回课程页继续处理后续知识点。

右侧日志面板会显示当前步骤、模型请求、参考答案命中和提交状态。

## 答案策略

脚本不是单纯每次都问 AI，而是采用“AI 回答 + 参考答案沉淀”的方式：

- 首次遇到题目时，主要由 AI 根据题干、题型和选项生成答案。
- 提交后，脚本会从解析页读取官方参考答案并按章节保存。
- 后续遇到相同题目时，优先使用已保存的参考答案。
- 选择题会参考同章节的相关历史题目，帮助模型判断。
- 填空题只使用完全匹配的历史题目，避免相似题误导答案。

随着同一章节运行次数增加，已保存的参考答案会逐步增多，相同题目的稳定性会更好。

## 注意事项

- 第一次运行某个章节时，准确率主要取决于模型能力。
- 如果页面长时间没有响应，可以刷新页面后重新开始。
- 自定义接口需要兼容 OpenAI 的聊天接口格式。
